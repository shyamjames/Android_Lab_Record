## Android Lab Record

### List of Questions
1. Create an app that displays name and email ID.
2. Create an app that takes marks of any three subjects from the user and displays its sum and average.
3. Create suitable apps to design the different layouts of Android Studio.
4. Develop a simple calculator app.
5. Create an app that changes text color when a button is clicked.
6. Create an app that displays Date and Time in different formats using Linear Layout.
7. Create an app that displays a Toast on the launch of an activity.
8. Create an app that creates an Alert Dialog Box.
9. Create a Greeting Card app that displays an image.
10. Create an app that plays audio when the **"Play Music"** button is clicked and plays a video when the **"Play Video"** button is clicked.
11. Create an app to demonstrate **Options Menu**.
12. Create an app that changes the background color of an activity using **Context Menu**.
13. Create an app to demonstrate how **RadioButtons** and **CheckBoxes** can be used in Android.
14. Create an app that launches a new activity on a button click.
15. Create an app that launches a new activity on a button click to display **student details** with **CheckBoxes** and **RadioButtons**.
16. Create an app that launches a new activity on a button click to calculate and display **employee details and salary**.
